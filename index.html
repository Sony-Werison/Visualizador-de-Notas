Não está carregando?

Pode ser necessário ativar o acesso ao proxy. Clique aqui para fazer isso e depois tente visualizar novamente.

Como obter as credenciais:

  1. API Key: Crie uma "integração interna" em notion.so/my-integrations. Copie o token "Internal Integration Token".
  2. ID da Página: Abra sua página no Notion. O ID é a parte final da URL (Ex: `...-PAGEID`). Você pode colar a URL inteira aqui.
  3. Permissão: Na página do Notion, clique nos 3 pontos no canto superior direito, vá em "Add connections" e adicione a integração que você criou.
